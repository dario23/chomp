Module chomp::primitives [−] [src]

Module used to construct fundamental parsers and combinators.

The Primitives trait allows access to the primitive methods of the Input trait. These methods are hidden in the documentation to make it easier to read the documentation since the methods are not useful when using the library or writing primitive parsers.
